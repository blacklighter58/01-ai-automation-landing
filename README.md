# AI Automation Landing

Первый учебный лендинг для услуги AI-автоматизации малого бизнеса.

## Что есть в проекте

- Адаптивная HTML-страница
- Современный дизайн на CSS
- Блоки с услугами и ценами
- Форма заявки
- JavaScript-обработка формы без перезагрузки страницы

## Технологии

- HTML
- CSS
- JavaScript
- Git
- GitHub
- Vercel

## Цель проекта

Проект создан как первая работа в портфолио в рамках обучения вайб-кодингу.
